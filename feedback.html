⚙ Подключение Google Apps Script

Страница работает через Google Apps Script — данные хранятся в вашей Google Таблице и отображаются здесь публично. Настройка делается один раз.

  1. Создайте новую Google Таблицу
  2. Откройте Расширения → Apps Script и вставьте код из файла apps_script.gs
  3. Нажмите Развернуть → Новое развёртывание → тип: Веб-приложение → выполнять как: Я → доступ: Все
  4. Скопируйте URL развёртывания, вставьте ниже и нажмите Сохранить
